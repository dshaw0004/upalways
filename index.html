UpAlways

UpAlways is a replicate of the uptimerobot. It is totally free to use. Currently it can only make http and https request. Still it is a great alternative of uptimerobot for making your Replit's api, telegram bot or discord bot always on.
Contact me to help me for adding new features to this site.

Note : By default UpAlways will make http(s) request to your api or bot in every 1 minute.
